Self-hosted badges

A shields.io-compatible static badge service powered by badge-maker, simple-icons and GitHub @primer/octicons.

Drop-in replacement for both:

  • https://img.shields.io/badge/...
  • https://custom-icon-badges.demolab.com/badge/...

Examples

/badge/Linux-FCC624?style=plastic&logo=linux&logoColor=black
/badge/Windows-0078D6.svg?style=plastic&logo=device-desktop&logoColor=white

Grammar: /badge/<label>-<message>-<color> (use _ for a space, __ for a literal underscore, -- for a literal dash). Query params: style, logo, logoColor, label, labelColor, color.
